# Generate Script Manual

Репозиторий для скриптов ручной выгрузки данных и сопутствующей документации.

## Структура проекта

- **Script/** — папка со скриптами (не коммитится в git, см. `.gitignore`). Здесь размещается скрипт загрузки профилей героев.
- **Docs/** — дополнительная документация. Подробное описание скрипта загрузки профиля героев: назначение, константы, функции, порядок использования — в файле [Docs/Скрипт_загрузка_профиля_герои.md](Docs/Скрипт_загрузка_профиля_герои.md).
- **log/** — каталог для логов (при наличии логирования в проекте).

## Скрипт загрузки профиля героев

Скрипт выполняется в консоли разработчика браузера на странице выбранного стенда (ALPHA или SIGMA), запрашивает профили по списку табельных номеров и сохраняет результаты в JSON-файлы.

Полное описание скрипта, всех переменных и функций — в [Docs/Скрипт_загрузка_профиля_герои.md](Docs/Скрипт_загрузка_профиля_герои.md). В документации не используются полные URL и реальные табельные номера; приведены маски и примеры вида `00001234`.
