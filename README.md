# Generate Script Manual

Репозиторий для скриптов ручной выгрузки данных и сопутствующей документации.

## Структура проекта

- **Script/** — скрипты для запуска через консоль разработчика на странице стенда.
  - Загрузка профилей героев: базовый `.js` (табельные из массива в коде) и `_from_file.js` (ТН из файла `.txt` или из массива).
  - Выгрузка файлов gamification: **`File_DB_Load_GP.js`** — POST `file-download`, панель с основными выгрузками, группами «Рейтинг» и «Заказы».
- **Docs/** — подробная документация по отдельным скриптам.
- **log/** — каталог для логов (при наличии логирования в проекте).

## Скрипт выгрузки файлов gamification (`File_DB_Load_GP.js`)

**Задача:** инициировать скачивание выгрузок (CSV и др.) с API gamification в браузере пользователя, используя сессию текущей вкладки (`credentials: "include"`), без вшивания куков и табельных номеров.

**Решение:** фиксированный список задач (основные три запроса + 10 вариантов рейтинга + 10 вариантов заказов); для каждой задачи — `fetch` POST с JSON-телом; ответ сохраняется как файл. Пакетные режимы идут **последовательно** с настраиваемой паузой. На странице показывается компактная панель: основные кнопки сверху, рейтинг и заказы **в двух колонках** рядом, затем «Скачать всё» и «Закрыть».

**Документация:** полное ТЗ по константам, задачам, функциям, UI и логам консоли — в файле  
[Docs/Скрипт_выгрузка_файлов_gamification_File_DB_Load_GP.md](Docs/Скрипт_выгрузка_файлов_gamification_File_DB_Load_GP.md).

**Краткий перечень сущностей в коде:**

| Имя | Назначение |
|-----|------------|
| `STAND`, `STAND_ORIGINS` | Выбор стенда (ALPHA / SIGMA) и базового origin |
| `DOWNLOAD_JOBS` | Три основные выгрузки (турниры, награды, статистика) |
| `RATING_GROUP_JOBS` | 10 POST на рейтинг (`businessBlock` + `timePeriod`) |
| `ORDERS_GROUP_JOBS` | 10 POST на заказы (`businessBlock` + `listType`, в т.ч. `NONSEASON`) |
| `getAllDownloadJobs`, `downloadOneJob`, `downloadJobsSequentially` | Сборка списка, один запрос, пакет с паузой |
| `downloadAllJobs`, `downloadRatingGroupOnly`, `downloadOrdersGroupOnly` | Сценарии кнопок |
| `startDownloadPanel` | Отрисовка панели (автовызов при вставке скрипта) |

**Запуск:** открыть нужный стенд в браузере → DevTools → Console → вставить содержимое `Script/File_DB_Load_GP.js` → Enter.

## Скрипт загрузки профиля героев

Скрипт выполняется в консоли разработчика на странице выбранного стенда, запрашивает профили по списку табельных номеров и сохраняет результаты в JSON-файлы.

Полное описание, переменные и функции — в [Docs/Скрипт_загрузка_профиля_герои.md](Docs/Скрипт_загрузка_профиля_герои.md). В документации не используются полные URL и реальные табельные номера; приведены маски и примеры вида `00001234`.
